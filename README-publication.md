# Publication GitHub Pages

1. Remplacer les fichiers du site GitHub Pages par le contenu du dossier `seo-site`.
2. Garder les images existantes a la racine du site : `florian.jpg` et `arnaud.png`.
3. Publier les fichiers suivants a la racine : `index.html`, toutes les pages `.html`, `style.css`, `sitemap.xml`, `robots.txt`, `CNAME`, `favicon.svg`, `og-image.png`, `og-image.svg`, `404.html`.
4. Ne pas supprimer le fichier `CNAME`. Il doit contenir `www.icaros-consulting.com` pour conserver le domaine personnalise.
5. Dans GitHub, aller dans `Settings > Pages` et verifier que la branche de publication pointe vers la racine du repository ou le dossier configure.
6. Apres publication, tester :
   - `https://www.icaros-consulting.com/`
   - `https://www.icaros-consulting.com/sitemap.xml`
   - `https://www.icaros-consulting.com/robots.txt`
7. Dans Google Search Console, ajouter la propriete `https://www.icaros-consulting.com/` puis soumettre `https://www.icaros-consulting.com/sitemap.xml`.
8. Utiliser l'outil d'inspection d'URL de Google Search Console pour demander une nouvelle indexation de la page d'accueil et des pages services principales.
